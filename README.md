# Vue dismiss
[![Build Status](https://travis-ci.org/dbrekalo/vue-dismiss.svg?branch=master)](https://travis-ci.org/dbrekalo/vue-dismiss)
[![NPM Status](https://img.shields.io/npm/v/vue-dismiss.svg)](https://www.npmjs.com/package/vue-dismiss)

Vue dismiss is a simple, reusable directive for executing closing logic when something outside component element is clicked
or when escape key is pressed. Applicable to dropdowns, modals and other types of menus. Weighs less then 1KB.
